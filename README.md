# 🏢 Admin Work Handler

AI-powered admin dashboard with 4 professional tools.

## ✨ Tools
- ✅ Task Manager — Organize and prioritize tasks
- 📧 Email Writer — Write professional emails
- 📊 Report Generator — Create structured reports
- 🗓️ Meeting Summary — Summarize meeting notes

## 🛠️ Tech Stack
- Python + Flask
- LangChain + Groq API
- HTML + CSS + JavaScript

## 🚀 How to Run
1. pip install -r requirements.txt
2. Add GROQ_API_KEY in .env file
3. python app.py
4. Open: http://127.0.0.1:5002
